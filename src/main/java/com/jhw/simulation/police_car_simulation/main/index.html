Resumen

El problema Máxima Cobertura se describe como la selección de una cantidad predeterminada de instalaciones entre las posibles alternativas, de modo que se maximice la cobertura sobre nodos de demanda a satisfacer. Varias de las aplicaciones de este problema son: la distribución de estaciones de patrullas de policía, distribución de ambulancias, ubicación de cámaras de seguridad, distribuciones de sucursales de tiendas y bancos, entre otras. Para trabajar este problema existe la herramienta MCLP-Tools, que representa datos y soluciones sobre un mapa, aun así, este no abarca la simulación del comportamiento de las estaciones, tanto fijas como en movimiento, en una situación requerida. Es por esto, que luego de implementar una solución a un problema de máxima cobertura, se necesita ver, lo más real posible, su efectividad en una situación específica, y una de las alternativas, es la Simulación Basada en Agentes. La simulación se utiliza en un número creciente de áreas gracias a su capacidad para hacer frente a una amplia variedad de modelos, y puede ayudar a identificar, y evaluar cuantitativamente soluciones alternativas. En este trabajo se emplea la Simulación Basada en Agentes dentro del problema de máxima cobertura en una situación previamente configurada, que le permita ver a un analista si es correcta la representación de dichas estaciones para hacer aún más efectivo su despliegue.
